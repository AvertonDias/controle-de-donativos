# **App Name**: UniteLedger

## Core Features:

- Consolidated Financial Ledger: A dynamic table displaying daily rows for Date, OMTJ, Congregation, and the auto-calculated Daily Sum.
- Native Add Contribution Modal: A streamlined input interface using the device's native calendar and numeric keypad for quick Worldwide Work and Congregation entries.
- Automated Running Totals: Real-time calculation of bottom-line monthly summaries for all donation categories, matching the layout of paper ledgers.
- Progressive Web App Shell: An installable PWA implementation providing a mobile-first experience with an offline-capable interface.
- Smart Projection Tool: An AI-powered feature that uses historical entries as a tool to forecast future totals and identifies trends in congregation contributions.

## Style Guidelines:

- Primary Color: Royal Indigo (#4A3AFF), selected to evoke feelings of trust, purpose, and community stability.
- Background Color: Ultra-soft Slate (#F5F7FA), a slightly desaturated cool tone to ensure high legibility and a clean 'digital paper' feel.
- Accent Color: Deep Ocean (#2D1E99), an analogous shade with higher contrast for emphasis on interactive buttons and primary action paths.
- A pairing of 'Alegreya' for table headers to provide a classic, reverent look, and 'PT Sans' for ledger entries to ensure clarity and precision in financial reporting.
- Fine-lined, minimalist iconography from a thin-stroke set to keep the focus on numeric data rather than visual clutter.
- A spacious, card-based interface centered on the viewport to emulate the focused environment of a formal accounting record.
- Gentle fade-in transitions for the add-entry modal and slide-up animations for newly added table rows to maintain a serene user experience.
